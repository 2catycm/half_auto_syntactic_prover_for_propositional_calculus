### 具体功能模块的解释
1.恒真验证
使用模型论手段验证是重言式。如果不是，指出一个真值指派
2.子公式分析
输出到 子公式工序里面
3.翻译
输入英文的意思，匹配翻译成单符号语言
4.一级带入。（第一广度）
子公式作为数据库，
打印出三个公理模式的，2个或三个字母的，每个字母是子公式数据库中的每一个子公式的 所有公理
这是笛卡尔积。
输入到 一级公理。
